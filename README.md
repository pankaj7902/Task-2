Task Overview
This project demonstrates sales data analysis using Excel Pivot Tables.  
The goal is to summarize data, identify patterns, and generate business insights.

Tools Used
- Microsoft Excel  

Dataset
Superstore Sales Dataset

Steps Performed
1. Cleaned dataset (removed blanks, fixed date formats)
2. Converted raw data into Excel Table
3. Created Pivot Tables for:
   - Total Sales by Category
   - Sales by Region and Segment
   - Profit Margin analysis
4. Added slicers for interactive filtering
5. Created Pivot Charts for visualization
6. Wrote summarized business insights
7. Exported final report as PDF

Key Insights
- West region generates the highest sales
- Technology category contributes maximum revenue
- Consumer segment dominates total sales
- South region shows lower performance
- Profit margins vary significantly across regions
